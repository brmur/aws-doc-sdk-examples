# Amazon S3 Directory Bucket Basics for the SDK for JavaScript (v3)

## Overview

This example demonstrates how to use the AWS SDK for JavaScript (v3) to work with Amazon Simple Storage Service (Amazon S3) directory buckets. 

_S3 Directory Buckets are designed to store data within a single AWS Zone. Directory buckets organize data hierarchically into directories, providing a structure similar to a file system._
